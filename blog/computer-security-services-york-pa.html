How is EDR different from regular antivirus? +

Traditional antivirus matches files against a list of known threats. Endpoint Detection and Response (EDR) watches how programs behave — so it can catch brand-new malware that has never been seen before. If something on your computer starts acting like ransomware (encrypting lots of files fast), EDR shuts it down even if it's not in any virus database yet.
